Домашнее задание к уроку 1

Задание 1

Задать температуру в градусах по Цельсию. Вывести в alert соответствующую температуру в градусах по Фаренгейту. Подсказка: расчет идет по формуле Tf = (9 / 5) * Tc + 32, где Tf — температура по Фаренгейту, Tc — по Цельсию.

Задание 2

Объявить две переменные: admin и name. Записать в name строку "Василий"; Скопировать значение из name в admin. Вывести admin (должно вывестись «Василий»).

Задание 3

Чему будет равно JS-выражение 1000 + "108"?

Объяснение: Складываем разные типы. Один из операндов является строковым. Поэтому приводим второй операнд (число) к строке. Получается конкатенация строк.

Задание 4

Самостоятельно разобраться с атрибутами тега script (async и deffer).

Объяснение: deffer - аттрибут тега script, который откладывает выполнения скрипта до полной загрузки страницы. Пример - если в скрипте используется какой-то элемент, который ещё не был инициализирован на странице, то будет ошибка. И необходимо использовать аттрибут deffer

async - аттрибут тега script, при использовании которого скрипт по возможности выполняется асинхронно. То есть скрипт выполняется без ожидания загрузки и отображения веб-страницы. В то же время и страница не ожидает выполнения скрипта, а просто продолжает загружаться. Пример - в скрипте выполняется какой-то prompt или alert. Без аттрибута async загрузка веб-страницы не будет продолжаться, пока мы не закроем окно alert или prompt.
